# Todo App Backend with Node.js, Express, and MongoDB as database

-This is the backend component of a Todo App built with Node.js, Express.js, and mongoose.

-It provides APIs for creating tasks, deleting tasks, and gettasks() tasks from the database.

-This backend serves as the data source for the frontend component of the Todo App.

-Server is hosted on Render and frontend is hosted on Netlify.
